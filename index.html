Who I Am

My name is Gennaro Rubino and I am a software engineer. I graduated December 2020 from Capitol Technology University with cum laude honors in a bachelor's degree of Computer Science. My elective studies focused on both full stack web development and electrical engineering. If you are interested in some of my projects please keep scrolling or visit my github repository.


My interests involving programming and electronics include ROM hacking, pixel art, electronics modifications, and homebrew.


Before receiving my degree, I've had a background with art, technical rescue training, and an electricians' apprenticeship.

My Projects

Gym Journal

For my undergrad capstone I developed a dynamic web application for users to record and track fitness progression in training programs for running, weightlifting and pull ups. This project features log in credentials and authentication, reading from and writing to a persistent database, HTML view templating and AJAX.

Gym Journal Android App

An android predecessor to my Gym Journal web application. User's metrics for running, conjugate method power lifting and the "Insanity" fit test is persistently stored so that navigation or closing the application is not lost. These stored records can be navigated through to review fitness progression and comparisons to dictate improvements can be made.

Asteroids JS

The 1979 arcade game by Atari, Asteroids, recreated in javascript. All objects on screen are rendered through the use of HTML5 canvas elements. Sound and UFOs have not been added to the project.
